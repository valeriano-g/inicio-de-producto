Introduction

Python is a high-level and easy-to-learn programming language. It was created in 1991 by Guido van Rossum and has since gained great popularity in different fields, such as data science, web development and desktop application development.

One of the main advantages of Python is its clear and concise syntax, which makes it ideal for beginners in programming. It also has a large number of libraries and frameworks that facilitate the development of applications in different areas. Python has a large number of libraries that can be used in different projects. Some of the most common libraries include:

  • NumPy: Provides support for working with arrays and data arrays in Python.
  • Pandas: provides tools for data analysis and manipulation in Python.
  • Matplotlib: allows you to create graphs and data visualizations in Python.

Variables

In Python, variables are used to store values. You can declare variables by assigning a value to a name. For example:

x = 10 y = "Hello, World!"
Declaring variables

To declare a variable in Python, you can simply assign a value to a name. For example:

x = 10 y = "Hello, World!"
Variable scope

In Python, the scope of a variable refers to the part of the program where the variable can be accessed. There are two types of variable scope in Python:

  • Global scope: A global variable is one that is defined outside of any function or class, and can be accessed from anywhere in the program.
  • Local scope: A local variable is one that is defined inside a function or class, and can only be accessed from within that function or class.
Global variables

To declare a global variable in Python, you can use the "global" keyword before the variable name. For example:

def my_function(): global x x = 10

This will make the variable "x" global, and it can be accessed from anywhere in the program.

Function declarations

In Python, you can define a function using the "def" keyword, followed by the function name and the function's parameters. For example:

def my_function(x, y): return x + y

This function takes two parameters, "x" and "y", and returns their sum.
